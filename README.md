# Mi Portafolio Personal

## 📌 Descripción del Proyecto

Este trabajo práctico consiste en el desarrollo de una Landing Page de Portafolio Personal utilizando HTML y CSS. El objetivo es aplicar buenas prácticas de estructura semántica, estilos modernos, diseño responsivo y organización del contenido, incluyendo secciones como presentación personal, proyectos, habilidades, contacto y contenido adicional.
